perlyoutube-dl
==============

Version 1.0

Perl Wrapper for the youtube-dl executable

# Installation

perl Makefile.PL
make
make test
make clean
make install

# Dependencies

## Non-Perl
youtube-dl

## Perl
Moose
IPC::Open3::Simple
Test::More
Carp
